This bundle generates Liquibase files :

 - "liquibase.properties"  
   The Liquibase configuration file  
   
   
 - "changelog-master.yml"  
   The master file for change-log  
   
   
 - "change-001.sql"  
   The first changeset with SQL scripts and rollback
